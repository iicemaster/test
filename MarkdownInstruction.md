# Инструкция для работы с Markdown

## Выделение текста

Для выделения текста курсивом необходимо по краям поставить зёздочки(*). Например, *вот так*. 

Для выделения текста полужирным, необходимо выделить его двойными звездочками (**).
Например, **вот так**.
