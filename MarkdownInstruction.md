## Списки
Для создания ненумерованного списка надо исользовать (*) перед текстом.
Например:

* Элемент 1
* Элемент 2
* Элемент 3
* Элемент 4
* Элемент 5

Для создания нумерованного списка необходимо пронумеровать пункты. 
Например:

1. Пункт 1
2. Пункт 2
3. Пункт 3
4. Пункт 4
5. Пункт 5
